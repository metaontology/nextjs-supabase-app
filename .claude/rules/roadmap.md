# Roadmap 규칙

## 📍 로드맵 파일 경로

- **파일**: `docs/ROADMAP.md`

## 📋 Task 포맷

### Task 제목

- 형식: `- **Task XXX: 작업명** - 우선순위`
- 번호: 3자리 숫자 (001, 002, 003...)

### 완료 표시

- Task 제목: `- **Task XXX: 작업명** ✅ - 완료`
- 하위 항목: `  - ✅ 항목명`
- Phase 전체 완료: `### Phase N: 제목 ✅`

## 📊 진행 상황 라인 포맷

- `**📊 진행 상황**: Phase N 진행 중 (X/전체 Tasks 완료)`

## 📅 날짜 포맷

- `**📅 최종 업데이트**: YYYY-MM-DD`
