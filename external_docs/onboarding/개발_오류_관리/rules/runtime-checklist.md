# Runtime Checklist — 런타임 동적 오류 관리

`docs/guides/create-quality-gate_v4.md` 오류 방지 3가지 방법 중 🌐 **Playwright MCP** 레이어입니다.

🛡️ Quality Gate가 정적 코드 품질을 차단하지만,
실제 브라우저에서의 동작 오류는 이 체크리스트로 검증합니다.

> 클로드가 구현 후 스스로 브라우저를 열어 검증 — 사람이 확인하지 않아도 동작 보장

## 실행 시점

아래 변경이 포함된 작업 완료 전 Playwright MCP로 반드시 검증한다.

| 변경 유형               | 검증 대상                |
| ----------------------- | ------------------------ |
| Supabase 쿼리 추가/변경 | 데이터 CRUD 동작         |
| 인증 플로우 변경        | 로그인/로그아웃/리디렉션 |
| 새 Route Handler 추가   | API 응답 정상 여부       |
| 미들웨어 수정           | 보호 라우트 접근 제어    |
| 새 페이지/핵심 UI 구현  | 사용자 플로우 전체       |

## 검증 방법

ROADMAP.md의 테스트 체크리스트 시나리오를 Playwright MCP로 직접 실행한다.
브라우저 제어 방법 → `playwright.md` 참조

## 통과 기준

- 테스트 시나리오 정상 동작
- 브라우저 콘솔 오류 없음
- 기존 기능 회귀 없음
